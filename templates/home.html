plasmapo : plasmid-to-map-o

Two columns are required for features: type and name. Currently, promoter (p), cds (c), fusion (f) or terminator (t) features are supported. Features can be repeated any number of times Features are drawn by input order.

Colour can be specified for cds features in a 3rd column. 1 - 9 numbers map to different colours.

To retrieve vector graphics, just print to PDF through your browser after the plasmid map is rendered.
